# rn-sms-retriever

React Native implementation of Android SMS Retriever

## Installation

```sh
npm install rn-sms-retriever
```

## Usage

```js
import RnSmsRetriever from "rn-sms-retriever";

// ...

const result = await RnSmsRetriever.multiply(3, 7);
```
